# 🔬 VeriLens AI
### *See through misinformation — instantly.*

A **hackathon-grade**, production-ready fake news detection system combining:
- **Hybrid AI**: Logistic Regression (calibrated) + OpenAI GPT reasoning
- **Multi-source fact-checking**: Wikipedia + web evidence + semantic similarity
- **Premium Streamlit UI**: Dark theme, animated pipeline, confidence gauge

---

## 🏗️ Architecture

```
Input Text
    │
    ├─► ML Pipeline (40%)
    │     TF-IDF (word + char n-grams)
    │     Logistic Regression
    │     Platt-scaled confidence
    │
    ├─► LLM Reasoning (40%)
    │     GPT-4o-mini analysis
    │     Claim type detection
    │     Red flag identification
    │     (heuristic fallback if no API key)
    │
    └─► Evidence Check (20%)
          Wikipedia semantic search
          Web snippet retrieval
          TF-IDF cosine similarity
          Contradiction detection

FINAL_SCORE = (ML × 0.4) + (LLM × 0.4) + (EVIDENCE × 0.2)
```

## ⚡ Quick Start

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Set OpenAI key (optional but recommended)
export OPENAI_API_KEY="sk-..."

# 3. Generate training data
python generate_dataset.py

# 4. Train model
python train_model.py

# 5. Launch app
streamlit run app.py
```

## 🎯 Key Improvements Over v1

| Feature | v1 (Original) | v2 (VeriLens AI) |
|---------|---------------|------------------|
| Model | LR + word TF-IDF | LR + word+char TF-IDF + Platt calibration |
| Verification | Keyword overlap | Semantic cosine similarity + contradiction detection |
| LLM | None | GPT-4o-mini with structured JSON output |
| Decision | ML only | Weighted hybrid (ML + LLM + Evidence) |
| Confidence | Raw sigmoid | Temperature-scaled + Platt calibration |
| UI | Basic Streamlit | Animated dark UI with gauge, tabs, pills |
| Dataset | 100 samples | 600+ diverse template-generated samples |

## 🔑 Environment Variables

| Variable | Description |
|----------|-------------|
| `OPENAI_API_KEY` | OpenAI API key for GPT reasoning (optional) |

Without `OPENAI_API_KEY`, the system runs a high-quality heuristic fallback for the LLM layer.
